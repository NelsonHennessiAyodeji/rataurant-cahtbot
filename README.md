# Restaurant ChatBot

A restaurant ordering chatbot with session-based ordering and Paystack payment integration.

## Features

- Session-based user management (no authentication required)
- Interactive menu selection
- Order management (place, view, cancel orders)
- Order history
- Paystack payment integration
- Order scheduling option
- Responsive chat interface

## Tech Stack

- Node.js
- Express.js
- Express Session
- Paystack API
- Vanilla JavaScript (Frontend)
- HTML/CSS

## Installation

1. Clone the repository:

```bash
git clone <repository-url>
cd restaurant-chatbot
```
